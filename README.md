# Practicando-Python-para-Data-Science-Challenge-Alura-Store
📌 Descripción del proyecto

Este proyecto forma parte del Challenge de Data Science de Alura Latam.
El objetivo es analizar el rendimiento de las cuatro tiendas de la cadena Alura Store para ayudar al Sr. Juan a decidir qué tienda debería vender para iniciar un nuevo emprendimiento.

A partir del análisis de datos de ventas, reseñas de clientes y costos de envío, se busca identificar la tienda menos eficiente y presentar una recomendación basada en datos.

🎯 Objetivo del análisis

- Analizar la información de ventas de las cuatro tiendas para:

- Evaluar el rendimiento financiero

- Analizar el comportamiento de los clientes

- Identificar productos y categorías más vendidas

- Comparar costos de envío

- Determinar qué tienda debería venderse

🏆 Resultados y conclusión

Después de analizar todos los indicadores, se identificó que la Tienda 4 presenta el menor rendimiento general en comparación con las demás tiendas.

Los principales factores observados fueron:

- Menores ingresos totales

- Menor rotación de productos

- No presenta ventajas significativas en calificaciones o costos de envío

- Por esta razón, se recomienda que el Sr. Juan venda la Tienda 4, ya que es la tienda menos eficiente dentro de la cadena.
